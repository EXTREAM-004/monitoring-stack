
# Grafana Dashboard Setup

1. Open Grafana:
   http://localhost:3000

2. Login:
   - Username: admin
   - Password: admin

3. Add Prometheus Data Source:
   - URL: http://prometheus:9090

4. Create Dashboard Panels:
   - app_requests_total
   - process_cpu_seconds_total
   - process_resident_memory_bytes

5. Save dashboard.
